About Exoplanet Hunter AI

Project Overview

Exoplanet Hunter AI is an advanced machine learning system developed for the 2025 NASA Space Apps Challenge. This project addresses the challenge "A World Away: Hunting for Exoplanets with AI" by creating an automated system to identify exoplanets from mission data.

With thousands of exoplanets discovered and terabytes of data collected by NASA's space missions, manual analysis is becoming increasingly impractical. This AI system democratizes exoplanet detection, making it accessible to researchers and space enthusiasts worldwide.

Data Sources

Kepler Mission

Duration: 2009-2018
Method: Transit photometry
Legacy: Over 2,600 confirmed exoplanets

The Kepler mission revolutionized exoplanet science by continuously monitoring 150,000 stars in a fixed field of view, detecting the tiny dips in brightness caused by planets passing in front of their stars.

K2 Mission

Duration: 2014-2018
Method: Transit photometry
Innovation: Extended Kepler's life with new strategy

After reaction wheel failures, K2 adapted by observing different fields along the ecliptic plane, discovering hundreds more exoplanets and studying various astronomical phenomena.

TESS Mission

Duration: 2018-Present
Method: Transit photometry
Coverage: Surveying the entire sky

The Transiting Exoplanet Survey Satellite (TESS) is surveying 85% of the sky, focusing on nearby bright stars where planets are easier to study with follow-up observations.

NASA Space Apps Challenge

This project was created for the 2025 NASA Space Apps Challenge, a global hackathon hosted by NASA. The challenge brings together teams from around the world to solve real-world problems using NASA's open data.

Challenge Category: A World Away: Hunting for Exoplanets with AI
Difficulty: Advanced
Date: October 4-5, 2025

Open Source

This project is built using open-source technologies and NASA's publicly available datasets. The code demonstrates best practices in machine learning, data preprocessing, and web development.

Key Features:

  • Multi-mission data integration and harmonization
  • Advanced ensemble learning techniques
  • SMOTE for handling imbalanced datasets
  • Interactive web interface with real-time predictions
  • Comprehensive model evaluation and statistics
  • Batch processing capabilities for large datasets
